# 入门操作流程

初始使用git对一个目录进行版本控制需要以下步骤：

- 进入要管理的文件夹

- 执行初始化命令

  ```dockerfile
  git init
  ```

- 查看目录下的文件状态

  ```dart
  git status
  注：新增文件或被修改的文件颜色是红色
  ```

- 管理指定文件（红变绿）

  ```
  git add 文件名
  git add .
  ```

- 个人信息配置：用户名，邮箱【一次即可】

- 生成版本

  ```
  git commit -m'版本描述'
  ```
